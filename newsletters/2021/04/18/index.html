I also put a lot of time into getting my GitHub user information serverless function to display a users GitHub contributions heat map.

Check it out!

You can make your own heatmap with it, the serverless function takes in parameters for a user and a year, here's what the url looks like for my 2021 stats: 

https://ghui.vercel.app/heat.png?username=spences10&year=2021

Replace my username spences10 with yours to take a look!
