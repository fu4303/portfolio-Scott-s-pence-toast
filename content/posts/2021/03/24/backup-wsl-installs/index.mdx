I have always liked using Windows Subsystem for Linux, I'd routinely
trash installs then reinstall them from the Microsoft store

```bash
wsl --export Fedora-33 Fedora-33.tar
```

```bash
wsl --import Fedord-33-ms Fedora-33-ms Fedora-33.tar
```
